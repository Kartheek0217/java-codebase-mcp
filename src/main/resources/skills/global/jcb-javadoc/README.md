# jcb-javadoc

Terse method-level Javadocs for service, repository, and resource packages.

## What it does

Generates ultra-compressed method Javadocs adhering to specific project templates. Differentiates between service/serviceImpl/repository methods and resource endpoint methods. Preserves strict tags (`@implNote`, `@param`, `@return`, `@exception`, `@author`) without prose bloat.

## How to invoke

```
/jcb-javadoc
```

Also triggers on phrases like "generate javadoc", "add javadocs".
